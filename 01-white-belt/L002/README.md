# Lesson 002 — What is a Method? ⬜ White Belt

**Video**: (link once live) · **Channel**: [@CSharpSensei](https://youtube.com/@CSharpSensei)

A method is a named, reusable block of code — it takes input, does work,
and can hand back a result.

## Run it

```bash
dotnet run
```

Expected output:

```
Total: 15
Bulk order: 480
```

## This week's drill 🥋

Find **one place in your own code where you've copy-pasted logic** and turn
it into a method. Give it a name that says exactly what it does.
